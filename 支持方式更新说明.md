## 🔄 后续维护

### 1. 链接更新
如需更新支持链接，只需修改HTML中的href属性：
```html
<a href="新的链接地址" target="_blank" class="donation-btn coffee-btn">
```

### 2. 样式调整
可以通过修改CSS变量来调整按钮颜色：
```css
.coffee-btn {
    background: linear-gradient(135deg, 新颜色1, 新颜色2);
}
```

### 3. 内容更新
支持方式的标题和描述可以直接在HTML中修改。
